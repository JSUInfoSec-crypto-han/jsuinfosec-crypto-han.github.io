# JSUInfoSec Crypto Teaching Portal

This repository publishes the organization-level GitHub Pages site for `JSUInfoSec-crypto-han`.

Site structure:

- `/` course portal homepage
- `/classical-crypto/` classical cryptography learning site
- `/cross-learning/` cryptography and information theory cross-learning site
- `/announcements/` announcements and update notices

The two teaching subsites are built from the source repository:

- `https://github.com/JSUInfoSec-crypto-han/cmijohnson-crypto-learning`

GitHub Actions clones the source repository, builds both frontends, and publishes the final static site through GitHub Pages.
